# 🛡 NetSys AI

> **Analyze. Detect. Protect.**
> Paste a URL → Get a security verdict in seconds.

---

## 🚀 Overview

**NetSys AI** is a modern cybersecurity platform that analyzes digital threats such as phishing, malicious links, and vulnerabilities in a **simple, fast, and accessible way**.

Designed to scale from a **minimal URL analyzer** into a full **AI-powered cybersecurity ecosystem**.

---

## ⚡ How It Works

```
User Input
   ↓
URL Analysis
   ↓
Risk Scoring
   ↓
Result
```

**Output:**

- ✅ Safe
- ⚠ Suspicious
- ❌ Dangerous

---

## 🔍 Features

- 🔗 URL analysis (phishing detection)

- 🛡 Basic vulnerability detection
